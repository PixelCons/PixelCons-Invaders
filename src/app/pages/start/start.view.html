This open platform was started in 2018 to allow any artist to easily make digital collectibles within a shared ecosystem. Every PixelCon is unique and it's ownership is secured on the Ethereum blockchain. If you want to create and trade PixelCons, you must first make an Ethereum account. 2. Install MetaMask

Your browser now needs to be able to connect to the Ethereum blockchain using a browser extension called MetaMask. Once installed, click on its icon and proceed to set up your Ethereum account.
2. Connect to Ethereum

Your browser now needs to be able to connect to the Ethereum blockchain. Please make sure your browser is providing the Ethereum JavaScript API to loaded web pages.
3. Explore/Manage Account
